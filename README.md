# KtExtensionAndroid
An extension library of Kotlin especially designed for Android platform

### Extended Class
- Android
    - android.app
        - `Activity`
    - android.graphics
        - `Bitmap`
        - `Drawable`
    - androidx.palette
        - `Palette`
- Kotlin
    - `Array`
    - `ByteArray`
    - `List`
- Java
    - `ArrayList`
    - `File`
    - `InputStream`
- More Extension is on the way...
